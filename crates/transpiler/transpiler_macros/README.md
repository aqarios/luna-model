# lunamodel-transpiler-macros

Procedural macros for the LunaModel transpilation stack.

`lunamodel-transpiler-macros` is an **internal sub-crate** of the [LunaModel](https://github.com/aqarios/luna-model)
library.

**Important Note**: This crate is **not intended for external usage**. Please refer to the main
[LunaModel crate](https://github.com/aqarios/luna-model) for intended usage.
